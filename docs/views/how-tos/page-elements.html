How to style page elements

The NHS.UK digital service manual provides lots of CSS classes for styling page elements, so you should not need to write as much of your own Sass or CSS.

You can change things like the font size or the spacing between elements using the utility classes.

Explore the NHS.UK digital service manual to see what classes are available and how to apply them.
